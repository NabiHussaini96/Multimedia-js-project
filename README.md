#  Multimedia & Avancerad JavaScript

##  Beskrivning
Detta projekt är en del av Aktivitet 3 och fokuserar på att integrera multimedia och interaktivitet i en webbsida med hjälp av HTML5 och JavaScript.

Projektet innehåller:
- En videospelare med inbyggda kontroller
- En ljudspelare som styrs med JavaScript
- En egen mediaspelare med play, paus och volymkontroll
- En interaktiv canvas där en cirkel ändrar färg

---

##  Tekniker som används
- HTML5 (`<video>`, `<audio>`, `<canvas>`)
- JavaScript (DOM manipulation och HTMLMediaElement API)
- Canvas API

---

##  Funktioner

###  Video
Videon är inbäddad med HTML5 `<video>`-element och innehåller standardkontroller.

###  Ljud + Custom Player
Ljudet spelas upp via `<audio>`-element men styrs med egna knappar:
- Spela
- Pausa
- Justera volym

Detta görs med JavaScript och `HTMLMediaElement API`.

###  Canvas
En canvas används för att rita en cirkel.  
När användaren klickar på en knapp ändras cirkelns färg slumpmässigt.
